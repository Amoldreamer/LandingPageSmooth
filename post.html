E-commerce System

This was a project I started back in november 2019 while I was learning PHP and Laravel. I was the sole creator/developer and thus was responsible for responsibilities like analysis, designing and coding.

It's a website divided into two parts - one for customer and the other for administrator. A customer can visit the website, register and login, select categories of items, make payments and logout.

Similarly, the administrator can also login and add or remove categories and items.

Technologies:
  • - PHP
  • - MySQL
  • - Laravel
  • - JavaScript
  • - AJAX
  • - HTML
  • - CSS
